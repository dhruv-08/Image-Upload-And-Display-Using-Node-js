# Introduction
In this project, you can upload image in mongodb database and retrieve it.<br>
# Dependencies
    * body-parser
    * cookie-parser
    * debug
    * express
    * express-session
    * http-errors
    * jade
    * mongodb
    * mongoose
    * morgan
    * multer
# Requirements
mongodb and node should be in your device
